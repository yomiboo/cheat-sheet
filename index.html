CSS Typography


Typography Properties

Property Name Description
font‐family Font Family used to change the font of an element
font‐weight Font Weight controls how bold or thin text appears
font‐style Font Style used to italicise text
text‐transform Text Transformation used to style text to appear in upper or lowercase
letter‐spacing Letter Spacing sets horizontal spacing between individual characters in an element
word‐spacing Word Spacing sets spacing between words
line‐height Line Height used to set how tall each line containing text should be
text‐align Text Align aligns text to its parent element

Typography Keyword Values

Keyword Value Linked Property Description
serif font‐family used as a final fallback font, for fonts with strokes on the ends of the main strokes of the letters
sans‐serif used as a final fallback font, for fonts with strokes on the ends of the main strokes of the letters
bold font‐weight makes font weight bold
normal maintains normal font weight; default setting
lighter makes the text one font weight lighter than the element's parent value
bolder makes the text one font weight bolder than the element's parent value
italic font‐style will make text appear in italics
normal will keep the text normal (without italics); default setting
uppercase text‐transform displays the selected text elements in uppercase letters
lowercase displays the selected text elements in lowercase letters
